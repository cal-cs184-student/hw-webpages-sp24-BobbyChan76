Overview:

In this project we implemented some of the geometric modeling topics we’ve learned in the course such as bezier curves/surfaces using De Casteljau subdivision and triangle meshes. With Bezier curves, the idea is that we want to be able to create a smooth curve from just some sample points but we also want to have a curve with C2 continuity. This same idea can be used to create a Bezier surface from a grid of sample points. With triangle meshes, we can now really start working with aspects of 3D modeling and render really cool shapes/objects. In this project we focused on implementing some basic operations with the Half-Edge data structure, such as edge flipping and splitting, to make easy alterations to our mesh. Using these operations we are able to implement subdivision on 3D meshes which allows us to increase the quality/smooth out our 3D model by using more triangles.

Part 6: Loop Subdivision for Mesh Upsampling:

  • Briefly explain how you implemented the loop subdivision and describe any interesting implementation / debugging tricks you have used.
  • To implement the loop subdivision I followed the outline in the upsample function’s comments. I started with a for loop over all the vertices in the mesh so I can set their isNew attribute to false, signaling that it's part of the original mesh, and storing what their updated positions will be in their newPosition field. Then I iterated over all the edges in the mesh so I can set their newPosition field to be what the position of the new vertices that split those edges will be. I then iterated over all the edges in the mesh that are not new and not connected to a new and old vertex (along the original edge but it was just created from the split call) so I can flip the edge. After the call from split, I set the newPosition field of the returned vertex to be the newPosition field of the current edge I’m using. Next I iterate over all the edges that are new (isNew=true) and flip them if they currently connect a new and old vertex. Finally I iterate over all the vertices and update their position to be the same value as their newPosition attribute.
